# OpenBible Project

Before working on this project, read the wiki in `openwiki/` for architecture context.

## Quick Reference
- **Build**: `./gradlew assembleDebug` (minSdk=29, targetSdk=34)
- **Database**: Room v8, prepopulated via `createFromAsset("databases/openbible.db")`
- **Key constraint**: Prepopulated DB is gitignored — generated by `data/import_bible.py`
- **Navigation routes**: All defined in `Routes.kt`, wired in `NavGraph.kt`
- **Content data**: JSON files in `assets/locations/` — locations, events, parallels, verse links

## Always Check
- Building on a clean machine requires running the data pipeline first
- Data file ID changes break FK relationships — add deprecation mappings if renaming
- FTS5 search uses `@RawQuery` + `SimpleSQLiteQuery` — ensure FTS syntax is sanitized
- Importers run once via SharedPreferences — new data in JSON won't re-import unless the flag is reset or versioned

## Content Schema Reference
- `locations.json`: 64 locations with id, lat/lng, modern_name
- `verse_links.json`: 394 word-boundary-matched verse references
- `events.json`: 102 events FK'd to locations.id
- `parallel_traditions.json`: 39 cross-cultural entries FK'd to location_events.id
